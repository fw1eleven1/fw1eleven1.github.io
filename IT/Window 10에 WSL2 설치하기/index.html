Window 10에 WSL2 설치하기
WSL
2022-04-07

** 현재 윈도우10의 버전이 20H1, 20H2, 21H1 이거나 2004 이상인지 확인한다. 아니라면 업데이트

  1. CMD 관리자 권한 실행

  2. 아래의 명령어 2개 실행

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

  1. 컴퓨터 재부팅

  2. WSL2 Linux 커널 업데이트

  3. WSL 버전을 2로 변경

wsl --set-default-version 2

  1. 마이크로소프트 스토어에서 사용할 리눅스 다운로드 (Ubuntu 등)

  2. 다운로드한 리눅스 실행

  3. 사용자 아이디/비밀번호 입력

  4. 설치된 리눅스와 WSL 버전 확인

wsl -l -v
  1. 버전이 1이라고 표시될 경우 2버전으로 다시 변환
wsl --set-version [리눅스명] 2

추가)

마이크로소프트 스토어에서 Windows Terminal 다운받아서 사용하면 CMD 보다 편함
